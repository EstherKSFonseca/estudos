Atributo rel (relationship) + EXTERNAL

Acesse um Link EXTERNAL NOFOLLOW, ou seja, é um link EXTERNO que você não quer que o Google siga.

Acesse um Link EXTERNAL NEXT, ou seja, para que o Google saiba que mesmo indo para fora do seu Site, está pagina ainda pertence ao conteúdo do seu site.

Acesse um Link EXTERNAL PREV, ou seja, para que o Google saiba que este link contém um conteúdo prévio ao seu site, como se fosse um tipo de "introdução" ao seu conteúdo.

Acesse um Link EXTERNAL AUTHOR, ou seja, para que o Google saiba que este link contém um conteúdo quer pertence ao mesmo Autor desse site.
